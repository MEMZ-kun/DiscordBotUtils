# Discord Bot Utility Framework

Python (`discord.py`) で高機能なボットを構築するためのユーティリティ群（ひな形）です。

---

## ⚠️ 注意事項

**本プログラムの利用には、以下の点に同意する必要があります。**

* **無断使用の禁止**
    トラブルを防ぐため、事前の相談なく本プログラムを使用しないでください。

* **品質に関する免責**
    作者はPythonに習熟していないため、コードに不具合が含まれる可能性があります。

* **無保証**
    事前の相談なく本プログラムを使用したことによっていかなるトラブルが発生しても、**作者は一切の責任を負いません**。また、質問も一切受け付けません。

* **再配布の禁止**
    いかなる理由があっても、事前の相談なく本プログラム（改変したものを含む）を配布することを固く禁止します。

## 🛠️ 必要なライブラリ

提示しているものは `pip` で環境を構築する例です。ご自身の環境に応じて変更してください。
